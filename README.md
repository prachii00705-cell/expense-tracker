# 💰 Expense Tracker

A responsive Expense Tracker built with **React + Vite** to help users manage expenses, analyze spending, and export reports.

## ✨ Features

- ➕ Add, edit, and delete expenses
- 🔍 Search, filter, and sort expenses
- 📊 Dashboard with analytics and pie chart
- 📁 Export reports as PDF and CSV
- 💾 Local Storage support
- 🔔 Toast notifications
- 📱 Fully responsive design

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- Chart.js
- React Toastify
- jsPDF
- jsPDF AutoTable
